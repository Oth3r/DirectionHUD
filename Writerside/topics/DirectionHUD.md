# DirectionHUD

The DirectionHUD command is can be used to execute any command in the mod, but is rarely used in favor of the more proprietary commands.

## Command

Syntax:

```shell
cmd [OPTIONS]
```

## Options

Describe what each option is used for:

-o, --open
: Opens a file.

-c, --close
: Closes a file.

-v, --version
: Displays version information.

-h, --help
: Displays help.
